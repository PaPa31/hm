If you are working with **macros** and [multi-command extension](../extensions/multi-command-extension), you should know VS Code commands.

## Find the name of the command you want to execute

1. Execute `Developer: Set Log Level...` and select `Trace` in the command palette.
2. Execute command of you want to know the name.
3. You can see the name of process in the output terminal `Log(Window)` (you can set the process for output on the right side of the output terminal).
