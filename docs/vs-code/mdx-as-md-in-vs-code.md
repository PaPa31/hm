title: mdx as md in VS-Code
date: 2021-03-21 15:13:22
---

To add the markdown functionality to `.mdx` files in `VS-Code`, you need to add this lines to `settings.json`:

```json title="settings.json"
"files.associations": {
    "*.mdx": "markdown"
},
```
