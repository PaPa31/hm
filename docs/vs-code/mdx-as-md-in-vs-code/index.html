mdx as md in VS-Code

To add the markdown functionality to .mdx files in VS-Code, you need to add this lines to settings.json:

settings.json
"files.associations": {
    "*.mdx": "markdown"
},

Source: stackoverflow.com: How to see preview of .mdx file as markdown in VSCode
